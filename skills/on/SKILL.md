# /bbuddy:on — Unmute the companion

Run this skill when the user types `/bbuddy:on`.

## Role

Call `bbuddy_unmute` to re-enable automatic reactions after a prior
`/bbuddy:off`.

## Call

```
bbuddy_unmute({})
```

Show a brief confirmation message from the tool response.
